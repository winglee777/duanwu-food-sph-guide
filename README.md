# 粽香启夏 · 端午食饮赛道爆品趋势

视频号小店 2025 端午周期（5月11日—6月7日）食品饮料赛道爆品洞察。

- **取数周期**：2025 年 5 月 11 日 — 2025 年 6 月 7 日（4 周端午周期）
- **数据来源**：视频号小店全域 TOP 商品（非全量）
- **品类范围**：10 大品类（粮油调味 / 生鲜 / 休闲食品 / 饮料冲调 / 乳品冷饮 / 传统滋补 / 酒类 / 宠物生活 / 保健食品·营养补充 / 茗茶）
- **已部署**：https://winglee777.github.io/duanwu-food-sph-guide/

## 数据加工流程

```bash
# 1. 原始数据放上一级目录
ls ../2025年视频号端午食品饮料商品总表.xlsx

# 2. 跑数据加工脚本（输出 data/records.json + data/stats.json）
python3 gen_618.py

# 输出预期：
# - 3211 条商品记录
# - 约 900 条经过品类二次校正
# - 10 大品类聚合 stats
```

## 品类清洗规则

`category-rules.json` 解决了"商家上架时勾错类目"的脏数据问题。例：

- 雪媚娘大福 / 月饼 / 卤味 → 自动从「粮油调味」纠正为「休闲食品」
- 牛羊肉水饺 / NFC 苹果汁中的"鱼/苹果"→ 不会误命中生鲜（被 priority 顺序保护）
- 复合词排除：休闲食品规则里的"猪肉"，遇到"猪肉粽"会被 exclude_keywords 跳过

修改规则后重跑 `gen_618.py` 即可生效，**不需要动 HTML**。
